# The ToDo

let us: Go! 2022 summer

## Features

- UIKit 으로 만들어진 ToDo 샘플 앱 입니다.
- UI는 스토리보드로 구성되었습니다.
- UI구성과정은 commit으로 표현하지 않았습니다.
- commit 단위로 코드가 진행되는 과정을 보시면 됩니다.
- Protocol 을 우선해서 만들고, 나중에 구현을 해가는 과정을 보여줍니다.
